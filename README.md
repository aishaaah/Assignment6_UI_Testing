This Maven project demonstrates automated UI testing using **Microsoft Playwright for Java**.  
The test suite validates navigation, search, add-to-cart, and checkout flow on the  
DePaul University Bookstore website.

---
 Tests Included
- `BookstoreFlowTest` — full e-commerce flow (search → cart → checkout → remove item)  
- `ExampleTest` — verifies basic page load and title retrieval  
- `SanityTest` — lightweight sanity check
